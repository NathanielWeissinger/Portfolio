P10: Merge Sort
Nathaniel Weissinger
12/09/2020


OVERVIEW:
Sorts lists that implement the IndexedUnsortedList interface,
using ordering defined by class of objects in list or a Comparator.
As written uses Mergesort algorithm.


COMPILING AND RUNNING:

From the directory of the program, to compile on Onyx, type:
$ javac SortTester.java

To run the programs, type:
$ java SortTester
